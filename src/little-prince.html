英语学习方法介绍

1. 学习英语常见问题
  • 听不懂 : 很多英语老师默认孩子学过中文语法，直接讲课。
  • 记不住 : 单词、语法太多，没时间复习。
  • 没意思 : 平时用不到，考试抱佛脚。
2. 立信英语特训的学习逻辑
  • 时间短 : 每天半小时，让改变悄然发生。
  • 方法正确 : 不再把英语拆分成单词和语法，而是综合学习，扩大知识面，更贴近新中考高考。
  • 项目制学习 : 通过每天翻译一篇文章，让单词和语法自然融入，不死记硬背，文章素材多样提高知识面。
  • 工具便捷 : 提供翻译、语音识别等工具，方便学习。
  • 进步可见 : 可以看到自己学习报告的前后对比，成就感强。
3. 训练原理
  • 沉浸式学习法
  • 间隔重复记忆法
  • 输出驱动学习法

学习建议

通过《小王子》学习英语的建议：

  1. 分章节学习 : 每天学习1-2个章节，循序渐进
  2. 角色扮演 : 尝试朗读不同角色的对话，提高口语
  3. 词汇整理 : 每章整理生词和短语，定期复习
  4. 主题讨论 : 思考每章的寓意，用英语表达自己的想法
  5. 结合立信方法 : 使用项目制学习法，每天翻译一段文本
