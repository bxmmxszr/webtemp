英语学习方法介绍

1. 学习英语常见问题
  • 听不懂 : 很多英语老师默认孩子学过中文语法，直接讲课。
  • 记不住 : 单词、语法太多，没时间复习。
  • 没意思 : 平时用不到，考试抱佛脚。
2. 立信英语特训的学习逻辑
  • 时间短 : 每天半小时，让改变悄然发生。
  • 方法正确 : 不再把英语拆分成单词和语法，而是综合学习，扩大知识面，更贴近新中考高考。
  • 项目制学习 : 通过每天翻译一篇文章，让单词和语法自然融入，不死记硬背，文章素材多样提高知识面。
  • 工具便捷 : 提供翻译、语音识别等工具，方便学习。
  • 进步可见 : 可以看到自己学习报告的前后对比，成就感强。
3. 训练原理
  • 沉浸式学习法
  • 间隔重复记忆法
  • 输出驱动学习法

六年级英语学习建议

针对六年级学生的英语学习建议：

  1. 系统复习 : 全面复习小学阶段学过的词汇和语法，查漏补缺，形成知识网络。
  2. 提升综合能力 : 重点加强阅读理解和听力综合训练，逐步提高处理较长文本的能力。
  3. 尝试写作 : 从简单的句子、段落开始，尝试用英语描述事物、表达观点。
  4. 运用高效方法 : 坚持使用立信英语特训的学习逻辑，如每天进行短文翻译，将知识融会贯通。
  5. 做好过渡准备 : 适当了解初中英语的特点和要求，为即将到来的学习阶段做好心理和知识准备。
